you can make the config:

$ git config --global user.name "your_name"
$ git config --global user.email "your_email@email.com"

you can generate SSH Keys:

1.check for SSH keys:
  $ cd ~/.ssh
  $ ls

2.if already had a key, backup and remove it; else to 3
  $ mkdir key_backup
  $ cp id_rsa* key_backup
  $ rm id_rsa*

3.generate a new SSH key:
  $ ssh-keygen -t rsa -C "your_email@example.com"

4.copy your SSH key, and add it to GitHub:
  $ clip < ~/.ssh/id_rsa.pub
  # copy the key to clipblard

5.goto https://github.com/ , add the key to your account

6.test everything out:
  $ ssh -T git@github.com
  # Attempts to ssh to github
  
  The authenticity of host 'github.com (207.97.227.239)' can't be established.
  RSA key fingerprint is ....
  Are you sure you want to continue connecting (yes/no)?

  # enter yes

  Hi username! You've successfully authenticated, 
  but GitHub does not provide shell access.
